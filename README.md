# Writing Samples — Terra Hyde

Technical and professional writing samples by Terra Hyde from her time at BigCommerce.

## About
This repository contains curated writing examples produced while working at BigCommerce. Samples include technical documentation, help and support articles, product copy, and internal communications that demonstrate clarity, audience-focused structure, and adherence to content standards.

## What’s included
- Technical how-tos and developer-facing guides
- End-user help articles and KB content
- Product and marketing copy
- Internal process and policy documents

## Usage & access
These samples are provided for review and portfolio purposes. Repository is public, and access to all files is allowed.

## Repository structure
- [API-Specs](https://github.com/ViciousElia/samples/tree/main/API-Specs) - Contains Swagger formatted YAML OpenAPI specifications
- [Guides](https://github.com/ViciousElia/samples/tree/main/Guides) - Contains guides and process documentation in MDX, DOCX, and other formats
- [Summaries-and-Values](https://github.com/ViciousElia/samples/tree/main/Summaries-and-Values) - Contains project summaries, value statements, concept overviews, and other non-complete looks at content
- [Diagrams](https://github.com/ViciousElia/samples/tree/main/Diagrams) - Contains diagrams meant for visualising data in several formats
- [Templates](https://github.com/ViciousElia/samples/tree/main/Templates) - Contains templates for reporting in multiple different contexts, from training plans to documentation requests
